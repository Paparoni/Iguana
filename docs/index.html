a List containing all of the elements added at the start of the list

origin
Example

myList.push("fizz");

myList.origin.forEach(function(element){ console.log(element) });
=> 1

▸ shuffle(num)

This function returns a random element from a List.

shuffle(num: number): any
Parameters
num (number) optional parameter, if given this function will return a new List with randomized elements this parameter will define the number of elements that will be in the shuffled list.
Returns
any:
Returns

Example

=> "foo"
// If shuffle is given a number as a parameter
// This will return a new a list with 3 random elements from the original list
